    points:

        - Recherche et validation d'adresses de chantier
        - Interface intuitive pour des utilisateurs non-techniques du BTP
solution:
    description: 'Nous avons développé une application web Symfony avec API Platform, offrant un espace dédié pour les mandataires et les sous-traitants, avec des intégrations tierces pour automatiser la saisie et fiabiliser les données.'
    points:
        - 'Espace mandataire : création de mandats, affectation de missions, suivi d''avancement'

        - API REST exposée via API Platform pour une architecture découplée
