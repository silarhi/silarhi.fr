Mise à jour technique de l'application de gestion de stock des articles industriels.

Développements techniques :

- Migration PHP 5.6 vers 8.0
- Utilisation du SDK SAP au lieu d'usage en direct via l'extension php `COM`
- Support de l'UTF-8 dans les différentes pages
- Utilisation de requêtes SQL préparées afin de prévenir les injections SQL
