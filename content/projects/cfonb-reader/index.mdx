## Context

[CFONB Reader](https://cfonb.adebiaye.tech/) est un outil de consultation de fichiers bancaires aux formats CFONB120 et CFONB240. Le code source de ce projet est disponible en [open-source](https://github.com/silarhi/cfonb-reader).
