## Context

Lanterne est un outil de veille politique basé sur l'agrégation des textes publiés par l'Assemblée nationale, le Sénat ainsi que sur les comptes Twitter des personnalités politiques françaises.

Le site permet de consulter les dernières publications sur de grandes thématiques (sécurité, éducation, santé, etc.), de voir les acteurs politiques impliqués sur ces sujets ainsi que leurs positions publiques.
