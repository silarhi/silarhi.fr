Maintenance et évolutions de l'application de gestion de supports multimédias pour Airbus.

Fonctionnalités ajoutées :

- Intégration de services tiers (Chili Publish, Keepeek, Salesforce)
- Mise en place de contrôle interne des coûts via un processus de validation multi étapes
- Module de partage de supports marketing
- Intégrations de moteur de rendu 3D via WebGL

Développements techniques :

- Implémentation de SSO via SAML (One Login)
- Passage du stockage sous S3
