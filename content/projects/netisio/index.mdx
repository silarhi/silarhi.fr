challenge:
    description: 'Le développement d''une plateforme de monitoring énergétique présentait des défis techniques liés à la collecte de données IoT en temps réel, la fiabilité des relevés automatiques et la restitution claire de volumes importants de données de consommation.'
    points:
        - Collecte fiable des données depuis des modules matériels IoT
        - Gestion de multiples capteurs par module (jusqu'à 30 compteurs eau/électricité)
        - Traitement et stockage de volumes importants de relevés
        - Restitution temps réel des consommations avec tableaux de bord
        - Processus d'appairage simplifié entre modules physiques et plateforme
solution:
    description: 'Nous avons développé une application web Symfony avec API Platform, intégrant la réception de données IoT, un système d''appairage par code couleur et des tableaux de bord temps réel pour le suivi des consommations.'
    points:
        - API REST via API Platform pour la réception des données des modules IoT
        - Système d'appairage par code couleur pour associer modules et plateforme
        - Tableaux de bord temps réel avec visualisation des consommations
        - Gestion multi-sites et multi-équipements pour les gestionnaires
        - Interface Twig responsive pour consultation sur tous supports
