Développement complet de l'API de gestion de licences et codes d'installation avec Symfony et API Platform.

Fonctionnalités développées :

- Création de l'API
- Création des entités Doctrine depuis une base PostgreSQL existante
- Gestion des codes d'installation
- Gestion des utilisateurs
- Vérification de sécurité liée à la détection d'un code d'installation déjà utilisé
- Fourniture d'une API documentée via Swagger
