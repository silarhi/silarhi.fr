Développement d'une plateforme de refacturation de bons de commandes avec Symfony, API Platform et React.

Fonctionnalités développées :

- Création du site (authentification, gestion des utilisateurs via invitation, révocation des accès)
- Moteur de synchronisation des commandes dans Boomerang
- Import et export de données asynchrone
- Module de dépôt sécurisé de documents
- Module de messagerie instantanée
