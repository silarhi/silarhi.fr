## Architecture

### Ingestion Multi-Sources
Workers Python récupèrent contenus :
- Articles presse politique
- Textes lois Assemblée/Sénat
- Tweets acteurs politiques

Messages déposés queue [RabbitMQ](/projets/tag/queues) pour traitement asynchrone.

### Workers PHP Elasticsearch
Workers [PHP](/projets/tag/php) consomment RabbitMQ et indexent dans [Elasticsearch](/projets/tag/elasticsearch) :
- Stockage articles/textes structurés
- Indexation full-text multilingue
- Enrichissements métadonnées (auteur, date, source)

### Veille Personnalisée Mots-Clés
Module alertes utilisateurs :
- Configuration mots-clés surveillance (ex: "réforme retraites", "écologie")
- Notification email/SMS nouveaux contenus matchant
- Filtres avancés (source, acteur, thématique)

### Scoring Pertinence IA
**Moteur pertinence intelligent** via back-office :
- Administration poids mots-clés par thématique
- Boost canaux officiels (Assemblée > Twitter)
- Machine learning améliore scoring selon engagement utilisateurs

### Abonnement Stripe
Monétisation SaaS via [Stripe](/projets/tag/stripe) :
- Paiements récurrents mensuels/annuels
- Gestion abonnements (upgrade/downgrade)
- Webhooks synchronisation statuts
- Conformité PCI DSS
