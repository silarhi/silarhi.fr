## Context

[Guidap](https://guidap.com/) est un logiciel Web de gestion d'établissements de loisirs. L'outil permet la gestion de vos réservations, votre planning et votre comptabilité.

## Realisations

### Création de Workers et Consumers

Date : 14/01/2021

Développements techniques :

- Utilisation de queue via AWS SQS
- Gestion de la requeue avec exponential back\-off
- Mise en place d'une "meta\-queue" pour le stockage des messages expirés après un certain temps
- Module de dispatch de message en fonction du nombre de tentatives
