## Contexte

[IRIS](https://www.biotech-dental.com/en/iris/) est un scanner intra-oral de nouvelle génération pour professionnels dentaires. Une plateforme web complète permet aux partenaires distributeurs de gérer leurs commandes, suivre les livraisons et accéder à la documentation technique directement depuis leur navigateur.

Nous intervenons sur ce projet en tant que partenaire technique pour assurer les évolutions fonctionnelles, les migrations technologiques et la maintenance applicative.

## Défis : Faire Évoluer une Application Médicale Critique

Notre accompagnement technique présente des défis spécifiques au secteur médical :

- **Fiabilité et sécurité** : Gestion de commandes de matériel médical nécessitant une traçabilité complète
- **Performance** : Interface réactive pour une utilisation quotidienne par les partenaires
- **Évolutivité** : Architecture capable d'évoluer avec les besoins métier de Scan4All
- **Expérience utilisateur** : Interface moderne et intuitive pour les professionnels de santé

## Notre Intervention : Évolutions et Modernisation Continue

Nous accompagnons l'évolution de la plateforme IRIS avec des interventions régulières sur le backend Symfony et le frontend React.

### Migrations et Modernisations Backend

Nos interventions sur l'API Symfony incluent :

- **Migration Symfony 4.4 → 6.2** : Modernisation du framework avec adoption de PHP 8.2
- **Refonte authentification** : Support multi-méthodes (SSO, Keycloak, Bearer, Header)
- **Intégration APIs tierces** : Connexion avec systèmes partenaires privés
- **Optimisation CI/CD** : Réduction du temps de tests de 15min à 1min30
- **Séparation des données** : Architecture multi-bases pour scalabilité

### Évolutions Frontend React

Les améliorations de l'interface React comprennent :

- **Migration React 14 → 15** : Adoption des dernières fonctionnalités React
- **Refonte graphique complète** : Nouveau design system moderne
- **Migration Tailwind CSS** : Remplacement de l'ancien framework CSS
- **Composants shadcn/ui** : Uniformisation des composants graphiques
- **Amélioration UX** : Optimisation du layout applicatif et navigation

### Maintenance et Support Continue

Nous assurons également :

- Correction de bugs et hotfixes en production
- Support technique pour nouvelles fonctionnalités
- Revue de code et optimisations de performance
- Veille technologique et recommandations d'évolution

## Résultats et Impact

Nos interventions ont permis d'améliorer significativement la plateforme :

- **Performance CI/CD** : Tests 10x plus rapides (15min → 1min30), déploiements accélérés
- **Modernisation technique** : Stack à jour (Symfony 6.2, React 15, PHP 8.2)
- **Expérience utilisateur** : Interface modernisée et uniformisée appréciée des partenaires
- **Scalabilité** : Architecture multi-bases prête pour la croissance internationale
- **Maintenabilité** : Code modernisé facilitant les futures évolutions

## Technologies Utilisées

- Symfony 6.x - Framework PHP pour l'API REST backend
- React 18 - Bibliothèque JavaScript pour l'interface utilisateur
- Tailwind CSS - Framework CSS utility-first pour le design
- PHP 8.2 - Langage backend moderne et performant
- MySQL - Base de données relationnelle pour la persistance
- Docker - Conteneurisation pour le déploiement
