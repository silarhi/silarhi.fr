## Contexte

IRIS Web est l'interface utilisateur web du scanner intra-oral dentaire IRIS développé par Scan4All. Application permettant aux professionnels de visualiser, analyser et partager scans 3D réalisés avec le dispositif médical.

## Défis

Interface [React](/projets/tag/react) 14 nécessitait modernisation complète : composants UI manquaient homogénéité visuelle et cohérence ergonomique. React 14 approchait fin de support, exposant à risques sécurité et empêchant optimisations performance. Design vieillissant ne reflétait plus standards interfaces médicales modernes.

## Solution Technique

### Migration React 19
Montée de version vers [React](/projets/tag/react) 19 apportant Server Components (chargement optimisé), rendu concurrent amélioré, et re-rendus optimisés automatiquement. Support long terme et sécurité renforcée.

### Système Design shadcn/ui + Tailwind
Adoption shadcn/ui couplée [Tailwind CSS](/projets/tag/tailwind-css) transformant approche développement UI. Composants shadcn offrent base accessible conforme WCAG, tandis que Tailwind permet personnalisation fine identité visuelle IRIS. Stack moderne accélérant développement et garantissant cohérence graphique.

### Refonte Layout
Nouveau layout optimisant workflows praticiens dentaires : navigation simplifiée, zones d'action identifiées, hiérarchie visuelle améliorée. Uniformisation composants créant expérience fluide et professionnelle.
