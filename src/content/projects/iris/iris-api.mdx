Refonte de l'API avec nouveaux endpoints, multi-authentification et séparation des bases de données.

Fonctionnalités ajoutées :

- Intégration d'API tierces privées
- Création d'endpoints API dédiés aux partenaires
- Refonte des différents supports d'authentification (SSO, KeyCloak, Bearer, Header)
- Séparation des données vers plusieurs bases

Améliorations techniques :

- Migration Symfony 4.4 vers 6.2
- Mise en place de CI
- Temps de tests réduits de 15min à 1mn30
