## Fonctionnalités

### Affichage Parcelles Leaflet.js
Cartographie interactive via Leaflet.js :
- Affichage parcelles cadastrales géoréférencées
- Couches cartographiques (satellite, plan, cadastre)
- Navigation zoom/pan fluide
- Tooltips informations parcelles (surface, propriétaire, zonage)

### Filtres Recherche Multi-Critères
Système filtrage avancé :
- **Surface** : min/max m² ou hectares
- **Type parcelle** : agricole, constructible, bâtie
- **Commune** : sélection multi-communes
- **Adresse** : recherche textuelle géocodée

Résultats temps réel sur carte.

### Pins Prospection Géolocalisés
Outil prospection commerciale :
- Placement pins personnalisés (prospects, clients, concurrents)
- Prospection par rue, quartier, ville
- Notes et statuts sur pins (à contacter, rendez-vous pris, client)
- Itinéraires optimisés entre pins

Fonctionnalité mobile (GPS terrain).

### Authentification Multi-Utilisateurs
Système utilisateurs isolés :
- Login sécurisé personnel
- **Cloisonnement données** : chaque user voit uniquement ses pins
- Gestion équipes (partage optionnel)

Confidentialité prospection garantie.
