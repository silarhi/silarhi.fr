## Context

Le PLM désigne ici l'écosystème d'applications internes utilisée pour la gestion de la durée de vie des produits \(Product Life Management\).

## Realisations

### Tierce Maintenance Applicative

Date : 01/02/2020

Ajout de fonctionnalités :

- Module d'export asynchrone de données
- Migration du code PHP 5.6 vers 8.2
- Modernisation de l'architecture de plus de 10 applications PHP
- Mise en place d'Elasticsearch pour l'indexation de 7M\+ documents
- Gestion multi\-langue des différents produits
- Création d'un outil de synchronisation de groupes IAM vers Active Directory

## Technologies

PHP, jquery, bootstrap, elasticsearch, symfony, mercure
