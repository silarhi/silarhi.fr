Ajout de fonctionnalités de sécurité et d'automatisation de la vérification des documents.

Fonctionnalités ajoutées :

- Vérification des numéros de téléphone via API SMS
- Mise en place de la double authentification
- Vérification automatisée de l'avis d'imposition via le site des impôts
- Génération d'un PDF reprenant toutes les pièces du dossier locataire
- Création d'un accès dédié pour les garants
- Améliorations ergonomiques (miniatures de documents, nombre de candidatures)
- Mise en place d'une API

Mises à jour techniques :

- Migration Symfony 4.3 vers 5.4
