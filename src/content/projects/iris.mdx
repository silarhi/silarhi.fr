## Context

[IRIS](https://www.biotech-dental.com/en/iris/) est un scanner intra\-oral à destination des partenaires médicaux tels les dentistes. Une interface Web permet aux partenaires de gérer leur commande depuis leur navigateur.

## Realisations

### IRIS API

Date : 16/10/2023

Évolutions apportées :

- Intégration d'API tierces privées
- Création d'endpoint API dédié à des partenaires
- Refonte des différents supports d'authentification \(SSO, KeyCloak, Bearer, Header\)
- Séparation des données vers plusieurs bases

Développements techniques :

- Migration Symfony 4.4 vers 6.2
- Mise en place de CI
- Temps de tests réduits de 15min à 1mn30.

### IRIS Web

Date : 12/06/2024

Évolutions apportées :

- Refonte graphique
- Mise à jour du layout applicatif
- Uniformisation des composants graphiques

Développements techniques :

- Migration react 14 vers 15
- Refonte des composants UI vers shadcn / tailwind
