## Context

Immobilus est une application Web permettant à des propriétaires de biens de digitaliser leur gestion immobilière.

Le site permet au propriétaire de  gérer leur immeuble, bien, locataires, locations, loyers, quittance et facture.

## Realisations

### V1

Date : 24/07/2019

Développement du coeur de l'outil de gestion :

- Mise en place de l'outil
- login / reset password / gestion des rôles
- Gestion des immeubles, bien, locations, locataires, documents
- Moteur de calcul des loyers locataires \(loyers mensuels, trimestriels, ...\)
- Gestion d'accès via délégations
- Mise en place de récupération d'opérations bancaires via EBICS

### V2

Date : 02/10/2020

Ajout de fonctionnalités :

- Double authentification
- Gestion des prorata de loyers \(locataire arrivé ou parti en cours de mois\)
- Gestion du dépôt de garantie
- Mise en place de relances automatiques
- Récupération des indices de loyer via API INSEE
- Moteur de recherche globale

Mise à jour technique :

- Migration FOSUserBundle
- Migration Symfony 4.3 vers 5.4

### V3

Date : 25/01/2021

Ajout de fonctionnalités :

- Intégration des factures, avis d'échéance, facture acquittées, quittances de loyer
- Moteur de rapprochement automatique d'opérations bancaires
- Gestion de la fin de location
- Grille de vetusté
- Envoi de données automatisée aux impôts
- Module de délégation d'accès pour les locataires
- Signature electronique des PDF générés par la plateforme
- Moteur de calcul des charges d'immeuble
- Génération automatisée d'extrait de compte mensuels / trimestriels / etc
- Branding des profils bailleurs \(personnalisation des couleurs\)
- Création d'accès spécifiques pour les partenaires \(banque, avocat, huissier\)
- Moteur de calcul d'augmentation des loyers
- Prise en compte de baux multi biens

### V4

Date : 17/02/2022

Ajout de fonctionnalités :

- Amélioration ergonomie \(sections d'aides sur champs complexes, tri & recherche dans toutes les parties\)
- Moteurs de vérification périodiques de validité des données \(adresse postale, email, numéro de téléphone\)
- Gestion des clés \(immeuble, partie commune\)
- Refonte technique
- Améliorations sur la récupération d'opérations bancaires via EBICS \(gestion multi banques, multi profils\)
- Module de messagerie bailleur / locataire. Messages pouvant être envoyés à tous les locataires d'un bailleur ou d'un immeuble entier.
- Visualisation des donnée d'audit enregistrées par l'outil.
- Améliorations sur la gestion de fin de location \(remboursement de dépôt de loyer, calcul des dates et modalités de préavis\)

Mises à jour techniques :

- Migration PHP 7.4 vers 8.0

- Migration Symfony 5.4 à 6.0

### V5

Date: 26/07/2022

Ajout de fonctionnalités :

- Intégration API dosloc
- Autocompletion des adresses postales
- Améliorations de l'onboarding des locataires \(message de bienvenu, informations de démarrage\)
- Mise en conformité RGPD sur la suppression et l'archivage d'informations
- Relance par SMS
- Intégration d'opérations bancaires au format OFX
- Export d'écritures comptables
- Auto génération de formulaires type Cerfa 10842\*06
- Module de suivi de litige des locations
- Module d'alerte en cas de changement d'informations des sociétés locataires.
- Intégration Solution SEPA
- Génération automatique des quittances de loyer
- Gestion des fichiers CAF \(création d'une [librairie](https://github.com/silarhi/caf-parser) de parsing de fichier CAF\)
- Module de site vitrine pour chaque bailleur

Mises à jour techniques :

- Migration Symfony 6.0 à 6.4

### V6

Date : 24/06/2025

Ajout de fonctionnalités :

- Gestion de remises exceptionnelles \(franchise de loyer, remise de loyer, etc\)
- Mise en place de l'archivage de locations
- Améliorations sur l'outil de rapprochement bancaires \(Éclatement en sous\-opérations, auto\-affectation de sous\-opérations\)
- Gestion du changement de périodicité des loyers en cours de location
- Prise en charge des observatoires de loyer pour envoi automatique de données
- Gestion de la vente des baux en cours de location
- Module de grilles de vétusté
- Module de facturation

Mises à jour techniques :

- Passage du stockage des fichiers via S3.

## Technologies
