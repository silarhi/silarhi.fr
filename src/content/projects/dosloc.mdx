## Context

Dosloc est une application Web permettant la saisie de son dossier locataire.

Le site permet au locataire de  Créer son dossier de candidature, Déposer ses pièces justificatives \(Carte d'identité, justificatifs de revenus, ...\) et Partager son dossier avec son bailleur.

Le site permet au bailleur de Centraliser la gestion de ses candidatures, Télécharger le dossier de candidature au format PDF, Préparer le bail une fois la candidature acceptée.

## Realisations

### V1

Date : 24/07/2019

Développement du coeur de l'outil de gestion :

- Mise en place de l'outil
- login / reset password / gestion des rôles
- Module locataire : Enregister ses informations de base, saisir des pièces justificatives.
- Partage du dossier locataire via un lien, email, sms.
- Système de révocation des liens de partage.
- Pour le bailleur :
    - La gestion de ses biens, des candidatures de locations,
    - Valider ou refuser la validité de certains documents.
    - Invitation candidat pour créer son dossier

### V2

Date : 02/10/2020

Ajout de fonctionnalités :

- Vérification des numéros de téléphone via API SMS
- Mise en place de la double authentification
- Vérification automatisée de l'avis d'imposition via le site des impôts
- Génération d'un PDF reprenant toutes les pièces du dossier locataire
- Création d'un accès dédié pour les garants
- Améliorations ergonomiques \(miniatures de documents, nombre de candidatures, ...\)
- Mise en place d'une API

Mises à jour techniques :

- Migration Symfony 4.3 vers 5.4

### V3

Date : 26/07/2022

Ajout de fonctionnalités :

- Ajout d'informations pour les garants
- Gestion de locataire ou garant sociétaire
- Module de recherche locataire
- Système de relance automatique
- Module de vérification des données de société \(mise à jour, expiration des informations, ...\)
- Suppression automatique du profil après 3 mois
- Gestion des informations pour préparation du bail après acceptation de la candidature locataire

Mises à jour techniques :

- Migration PHP 7.4 vesr 8.0
- Migration Symfony 5.4 vers 6.0
