## Contexte

Outil automatisation facturation connectant CRM HubSpot, facturation Zoho Invoice et pilotage Google Sheets pour entreprise services.

## Intégrations API

### HubSpot CRM
Connexion [API HubSpot](https://www.hubspot.com/) :
- Récupération deals gagnés automatiquement
- Import données clients (coordonnées, montants contrats)
- Synchronisation statuts opportunités

### Zoho Invoice
Intégration [API Zoho Invoice](https://www.zoho.com/invoice/) :
- Création factures automatiques depuis deals HubSpot
- Génération PDF factures conformes
- Envoi automatisé clients par email
- Suivi paiements et relances

### Google Sheets Pilotage
**Paramétrage dynamique via Google Sheets** :
- Taux TVA modifiables en temps réel
- Remises commerciales par client
- Conditions paiement personnalisées
- Mentions légales variables

Flexibilité sans redéploiement code.

## Moteur Calcul

### Calcul Automatisé Montants
Moteur calcul factures :
- Montant HT base deals HubSpot
- Application taux TVA (Google Sheets)
- Remises commerciales selon client
- Calcul TTC final automatique

### Gestion Acomptes
Module acomptes avancé :
- Factures d'acompte (30%, 50%, etc.)
- Factures solde automatiques
- Gestion refus empreinte bancaire (paiement manuel)
- Suivi multi-acomptes par projet

### Téléchargement Mensuel
Export automatisé factures :
- Génération ZIP toutes factures mois
- Organisation par client/date
- Transmission comptabilité simplifiée

## Résultats

- **Automatisation** : Zéro saisie manuelle factures (HubSpot → Zoho)
- **Flexibilité** : Paramètres modifiables Google Sheets sans dev
- **Conformité** : Factures légales générées automatiquement
- **Productivité** : Exports mensuels accélèrent comptabilité
