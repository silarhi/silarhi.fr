## Context

[L'agence](https://www.lagence.fr/) est un site de gestion pour agence matrimoniale. L'outil permet aux administrateurs de gérer les demandes d'inscription, faire le suivi et la recommandation de profils aux abonnés.

## Realisations

### V1

Date : 31/05/2025

Développement du site :

- Création du site \(authentification, mot de passe oublié, vérification de l'email / numéro de téléphone\).
- Inscription pour les abonnés : Saisie de leur profil amoureux, préférences.
- Moteur de recommandations : Calcul automatisé d'un score d'attractivité entre 2 membres pour un traitement manuel par l'administateur.
- Module de relances marketing : Emails automatisés selon différents critères \(anniversaire, rendez\-vous déclenché, abonnement expiré, ...\).

## Technologies

PHP, symfony, bootstrap, jquery, mysql
