## Contexte

Renfort assistance technique pour client Apside dans secteur agence web. Support développement multi-technologies et projets clients variés.

## Interventions Principales

### Maintenance Sites Existants
Maintenance et développement fonctionnalités sur sites production :
- [Symfony](/projets/tag/symfony) - Évolutions features, corrections bugs
- [Drupal](/projets/tag/drupal) - Modules custom, mises à jour
- [WordPress](/projets/tag/wordpress) - Plugins, thèmes personnalisés
- PHP natif - Applications legacy

### Applications Mobiles Twixl
Développement applications mobiles natives via framework Twixl :
- Publications digitales (magazines, catalogues)
- Contenus interactifs offline
- Distribution App Store / Google Play

### Développements Sur-Mesure
Sites web custom développés avec [Symfony](/projets/tag/symfony). Architecture robuste pour projets clients spécifiques.

## Résultats

- **Polyvalence** : Support multi-technologies (Symfony, Drupal, WordPress)
- **Mobilité** : Applications natives via Twixl
- **Réactivité** : Interventions rapides maintenance et corrections
- **Qualité** : Développements conformes standards web
