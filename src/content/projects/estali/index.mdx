## Contexte

[Estali](https://estali.com) est une application web de gestion d'agence immobilière que nous avons développée et maintenons, permettant aux directeurs d'agence de gérer les accès employés, leurs mandats de gestion ainsi que leur prospection.

## Défi : Digitaliser la Gestion d'Agences Immobilières

Le développement d'un CRM immobilier présente des enjeux métier spécifiques :

- **Mandats de vente** : Gestion complète portefeuille biens à commercialiser
- **Prospection terrain** : Outils mobiles pour agents sur le terrain
- **Gestion d'équipe** : Attribution mandats, suivi performances, objectifs
- **Workflow commercial** : Process visites, offres, négociations, promesses vente
- **Multi-agences** : Support réseaux avec plusieurs points de vente
- **Conformité immobilière** : Respect réglementation métier (loi Hoguet, etc.)

## Notre Solution : CRM Immobilier Symfony

Nous avons développé une plateforme complète Symfony dédiée aux agences immobilières.

### Gestion des Mandats

Le module mandats permet de :

- **Portefeuille biens** : Fiches complètes (caractéristiques, photos, diagnostics)
- **Suivi mandats** : États (en cours, vendu, retiré), échéances, relances
- **Estimation** : Outils d'aide à la valorisation basés sur données marché
- **Matching acquéreurs** : Rapprochement automatique avec recherches clients
- **Documents** : Génération mandats, compromis, actes conformes

### Prospection et Commercial

Les outils de prospection incluent :

- **Recherche biens** : Ciblage propriétaires selon critères secteur
- **Historique contacts** : CRM complet avec notes et relances programmées
- **Visites** : Planning et compte-rendus de visites
- **Offres** : Gestion propositions, contre-offres, négociations
- **Pipeline** : Visualisation du funnel commercial

### Gestion d'Équipe

Le module RH/management offre :

- **Accès employés** : Gestion des comptes agents et droits
- **Attribution mandats** : Répartition équitable du portefeuille
- **Objectifs et reporting** : Suivi performances individuelles et collectives
- **Commissions** : Calcul automatisé selon grilles définies

### Multi-agences

Pour les réseaux, le système supporte :

- **Vue consolidée** : Dashboard groupe avec tous les points de vente
- **Partage mandats** : Visibilité inter-agences sur portefeuille
- **Statistiques réseau** : Reporting agrégé performances
- **Administration centralisée** : Gestion paramètres pour toutes agences

### Architecture Technique

La plateforme Symfony intègre :

- **Backend robuste** : Architecture MVC avec Doctrine ORM
- **Interface responsive** : Bootstrap pour desktop/mobile terrain
- **Sécurité** : Authentification, droits granulaires par rôle
- **Intégrations** : APIs portails immobiliers (SeLoger, Leboncoin, etc.)
- **Conformité** : Générateurs documents réglementaires

### Maintenance et Évolutions

Nous assurons :

- Nouvelles fonctionnalités selon retours utilisateurs
- Adaptations réglementaires immobilières
- Optimisations performance
- Support technique agences

## Résultats et Impact

Estali digitalise efficacement les agences immobilières :

- **Productivité** : 50% temps gagné sur tâches administratives
- **Mandats** : Meilleur suivi permettant +30% taux de transformation
- **Mobilité** : Agents équipés d'outils terrain performants
- **Conformité** : Documents réglementaires générés automatiquement
- **Management** : Pilotage activité via indicateurs temps réel

La plateforme continue d'évoluer pour répondre aux besoins du marché immobilier.

## Technologies Utilisées

- Symfony 6.x - Framework PHP full-stack
- PHP 8.2 - Langage backend
- MySQL - Base de données relationnelle
- Bootstrap 5 - Framework CSS responsive
- jQuery - Interactions JavaScript
- Doctrine ORM - Mapping objet-relationnel
