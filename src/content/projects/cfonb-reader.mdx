## Context

[CFONB Reader](https://cfonb.adebiaye.tech/) est un petit outil de consultation de fichiers bancaires aux formats CFONB120 ET CFONB240. Le code source de ce projet est disponible en [open\-source](https://github.com/silarhi/cfonb-reader).

## Realisations

### CFONB Reader

Date : 10/07/2023

Développement de l'application :

- Utilisation de Symfony 6.4.
- Upload de fichiers CFONB
- Validation des données fournies \(format, type, taille maximale\)
- Visualisation des données sous forme de tableau
- Auto\-suppression des documents toutes les 4 heures

## Technologies

PHP, symfony, bootstrap, ebics
