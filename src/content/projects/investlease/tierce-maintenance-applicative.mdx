## Évolutions Principales

### Migration Symfony 5.4 → 6.4
Migration vers version LTS avec support long terme. Modernisation base de code et bénéfice des améliorations performance Symfony 6.x.

### Génération PDF WebKit
Refonte complète système génération documents contractuels. Passage à moteur WebKit pour rendus HTML/CSS complexes : contrats, tableaux amortissement, conditions générales. Qualité professionnelle et templates maintenables.

### Intégration API Scoring Financier
Connexion API partenaires financiers pour scoring crédit automatique. Évaluation en temps réel capacité emprunt candidats avec décision instantanée.

### Traitement Relevés Bancaires CFONB120
Intégration relevés bancaires format CFONB120 pour réconciliation automatique paiements et suivi échéances.

### Tests et CI/CD
- Mise en place tests unitaires et fonctionnels
- Pipeline CI/CD automatisé (intégration et déploiement continu)
- Outils audit données pour conformité réglementaire

### Préparation Marque Blanche
Architecture adaptée pour multi-tenancy : personnalisation identité visuelle, configuration métier par partenaire.
