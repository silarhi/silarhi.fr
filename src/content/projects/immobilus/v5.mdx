## Contexte

Immobilus V5 étend écosystème avec intégrations externes et automatisations comptables/juridiques avancées.

## Intégrations Externes

### API Dosloc
Connexion bidirectionnelle avec Dosloc (plateforme dossiers locataires SILARHI). Synchronisation candidatures validées vers Immobilus, création automatique locations et locataires. Workflow complet candidature → bail.

### Autocomplétion Adresses
Intégration API adresses françaises pour saisie assistée. Validation temps réel et normalisation format postal.

### Import OFX Bancaire
Support format OFX (Open Financial Exchange) pour import opérations bancaires. Complément EBICS pour banques sans protocole sécurisé.

## Automatisations Comptables

### Export Écritures Comptables
Génération fichiers export compatibles logiciels comptables (FEC, CSV personnalisés). Bailleurs transmettent données directement à experts-comptables.

### Solution SEPA
Intégration paiements SEPA : génération fichiers virements XML conformes, prélèvements automatiques loyers. Automatisation complète flux bancaires sortants/entrants.

### Génération Cerfa 10842*06
Auto-génération formulaire fiscal Cerfa 10842 (déclaration revenus fonciers) pré-rempli selon données plateforme. Gain temps déclarations fiscales.

### Quittances Automatiques
Génération automatique quittances loyer mensuelles. Envoi email locataires sans intervention bailleur.

## Fonctionnalités Métier

### Onboarding Locataires
Messages bienvenue automatisés nouveaux locataires avec informations démarrage (codes accès, contacts, règlement immeuble).

### RGPD
Mise en conformité : suppression et archivage automatisés données selon durées légales conservation. Droit à l'oubli respecté.

### Relances SMS
Extension relances avec canal SMS en complément email. Multi-canal augmente taux réponse.

### Module Litiges
Suivi contentieux locations : impayés, dégradations, procédures juridiques. Centralisation documents et échéances procédurales.

### Alertes Sociétés
Système surveillance modifications données sociétés locataires (changement dirigeants, adresse siège, radiation). Détection automatique via API entreprises.

## Résultats

- **Intégration** : Workflow candidature → location automatisé via Dosloc
- **Conformité** : SEPA, Cerfa, RGPD conformes réglementations
- **Productivité** : Exports comptables et quittances automatiques
- **Proactivité** : Alertes sociétés détectent risques en amont
