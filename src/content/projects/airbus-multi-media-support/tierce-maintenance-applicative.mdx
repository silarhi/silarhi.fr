## Intégrations SaaS Majeures

### Chili Publish
Intégration [Chili Publish](https://www.chili-publish.com/) pour édition documents marketing interactifs. Génération catalogues, brochures personnalisables par pays/langue.

### Keepeek DAM
Connexion [Keepeek](https://www.keepeek.com/) (Digital Asset Management) pour centralisation assets : photos avions, logos, vidéos. Bibliothèque média unifiée.

### Salesforce CRM
Synchronisation [Salesforce](https://www.salesforce.com/) : récupération données clients/prospects pour personnalisation supports marketing. Cohérence données CRM ↔ plateforme.

## Fonctionnalités Développées

### Contrôle Interne Coûts
Workflow validation multi-étapes pour dépenses marketing. Process approbation hiérarchique (manager → directeur → finance) selon montants. Traçabilité budgétaire complète.

### Module Partage Supports
Système partage sécurisé supports marketing avec partenaires, filiales, distributeurs. Permissions granulaires par région/entité.

### Moteurs Rendu 3D WebGL
Intégration moteurs 3D WebGL pour visualisation configurateurs avions dans navigateur. Personnalisation cabines, couleurs, configurations sièges en temps réel.

## Migrations Techniques

### SSO SAML OneLogin
Implémentation Single Sign-On via [SAML](https://www.onelogin.com/) avec OneLogin. Authentification unique employés Airbus (Active Directory). Sécurité et UX simplifiée.

### Stockage AWS S3
Migration stockage fichiers vers [AWS S3](/projets/tag/aws). Assets volumineux (vidéos 4K, modèles 3D, images HD) dans cloud scalable. Coûts réduits vs stockage dédié.
