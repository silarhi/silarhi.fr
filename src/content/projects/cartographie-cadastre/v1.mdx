Développement d'une Preuve de Concept (PoC) d'une application web de cartographie cadastrale en PHP 8.0.

Fonctionnalités principales :

- Visualisation interactive d'un cadastre à partir de données géospatiales (GeoJSON)
- Recherche de parcelles par identifiant cadastral (section + numéro)
