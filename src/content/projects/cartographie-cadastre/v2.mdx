Amélioration et extension de la Preuve de Concept (PoC) et passage à une version 2.0 de l'application avec React / Symfony / Api Platform.

Fonctionnalités ajoutées :

- Visualisation interactive avec library Leaflet.js
- Gestion des couches de données (parcelles, bâtiments, routes)
- Recherche avancée avec filtres (surface, type de bâtiment)
- API RESTful avec Symfony et Api Platform pour la gestion des données cadastrales
- Authentification et autorisation des utilisateurs
- Tests unitaires et fonctionnels pour assurer la qualité du code
- Documentation technique et utilisateur
