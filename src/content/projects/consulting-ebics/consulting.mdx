## Programme Formation

### Fonctionnement Global EBICS
Présentation protocole standardisé échanges bancaires sécurisés :
- Architecture client-serveur
- Chiffrement asymétrique (RSA)
- Signatures électroniques
- Normes françaises et européennes

### Initialisation EBICS
Protocole initialisation connexion bancaire :
- **INI** (Initialization) : Envoi clé signature électronique
- **HIA** (Host Initialization and Authentication) : Envoi clés chiffrement/authentification
- **HPB** (Host Public Bank keys) : Récupération clés publiques banque

Échange certificats sécurisé pour établir canal confiance.

### Récupération Relevés Bancaires
Ordres téléchargement fichiers (FDL) :
- Récupération mouvements quotidiens
- Formats supportés (CFONB, MT940, XML)
- Automatisation via CRON

### Parsing CFONB120
Traitement format CFONB120 (relevés bancaires français) :
- Structure fichier CFONB
- Extraction opérations (débits, crédits, virements)
- Import données comptabilité/ERP
